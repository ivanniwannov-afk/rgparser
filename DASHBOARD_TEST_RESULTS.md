## 12. 📝 Рекомендации для Пользователя

### Для запуска dashboard:

1. **Windows (через start_dashboard.bat)**:
   ```batch
   start_dashboard.bat
   ```

2. **Прямой запуск**:
   ```bash
   streamlit run dashboard.py --server.port 8501 --server.address localhost
   ```

3. **Открыть в браузере**:
   ```
   http://localhost:8501
   ```

### Для работы с конфигурацией:

1. Редактировать триггерные слова через UI
2. Добавить API ключи через UI
3. Настроить параметры очереди через UI
4. Все изменения сохраняются в config.json

### Для мониторинга:

1. Открыть страницу "Дашборд"
2. Нажать "Обновить" для актуальных данных
3. Развернуть секции юзерботов/чатов для деталей
4. Просмотреть последнюю активность в логе
